# Frequently Asked Questions

## General

**What is Neuro?**
Neuro is a macOS productivity app for focus tracking, distraction detection, app blocking, and social accountability.

**Is Neuro available now?**
The marketing site is live. The macOS app is currently in development.

**What platforms will Neuro support?**
macOS is the initial target platform.

## Website

**How do I run the website locally?**
```bash
cd website
npm install
npm run dev
```
Then open [http://localhost:3000](http://localhost:3000).

**How do I build the website for production?**
```bash
cd website
npm run build
```
