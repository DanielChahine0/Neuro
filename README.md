# Neuro

A macOS app for focus tracking, distraction detection, app blocking, and social accountability.

## Structure

- `website/` — Next.js marketing site (Next.js 16, Tailwind CSS v4, Framer Motion)
- macOS app (SwiftUI) and Firebase backend are planned

## Website

```bash
cd website
npm install
npm run dev      # localhost:3000
npm run build    # production build
npm run lint     # ESLint
```
